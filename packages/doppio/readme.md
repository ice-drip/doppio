# @kaffee/doppio

> Doppio is headless browser check util

## Installation

Install with npm

```bash
  npm install --dev @kaffee/doppio
```

Install with yarn

```bash
  yarn add --dev @kaffee/doppio
```

Install with pnpm

```bash
  pnpm add -D @kaffee/doppio
```

## Example
```typescript
import { Doppio } from "@kaffee/doppio";

Doppio.shut_up();

```
